## Installation notes

 * `mkdir /usr/src`
 * `sudo apt-get install openjdk-6-jre maven2 subversion xvfb libgl1-mesa-dri x11-xkb-utils xserver-xorg`
 * sudo apt-get install firefox-3.5
 * get saucelenium:
    * sudo git clone git://github.com/saucelabs/saucelenium.git
    * cd saucelenium
    * sudo ./build-all.sh
